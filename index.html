About

Webooks is an online book seller which acts as a broker for Amazon and Barnes and Noble. Project developed during Enterprise Management class at IST (Lisbon Tech). In here you can find:
  • A front-end developed in .ASP
  • Business processes for an online book retailer
  • Lots of webservices integration

Instructions

The project is no longer maintained, but the source code is publicly available in the hopes it will be useful for someone.
